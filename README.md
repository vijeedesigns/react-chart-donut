# React donut chart

This package will help you to install and work with stand-alone donut chart which is made in React.

## How to use

`npm i react-chart-donut`

## Usage

```js
import SVGDonut from 'react-chart-donut';

<SVGDonut data={[{
        color: '#673AB7',
        value: 20
    }, {
        color: '#4CAF50',
        value: 30
    }, {
        color: '#FF9800',
        value: 50
    }]} />
```
